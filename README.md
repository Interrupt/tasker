tasker
======

Test REST API using Java / Jersey, runs an embedded Grizzly web server.

Starts up on http://localhost:8080/tasker

# Endpoints:

## Get all tasks
GET /tasks

## Get one task
GET /tasks/{key}

## Create a new task
POST /tasks/{key}

## Update a task
PUT /tasks/{key}

## Delete a task
DELETE /tasks/{key}
